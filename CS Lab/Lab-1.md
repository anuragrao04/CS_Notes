# Unix Commands
-   Man
-   gedit
-   Ls ( -a -l  -i )
-   Cat
-   Cat >abc.txt (opens file)
-   Cat >>abc.txt (appending)
-   Cat -n abc.txt (line numbers)
-   Touch
-   Mkdir 
-   Cd
-   Pwd (prints working directory)
-   Rmdir (remove directory)
-   Cp
-   Mv
-   Chmod (to change permissions) u/g/o +rwx / -rwx
-   Chmod 755 abc.txt
